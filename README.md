<h3 align="center"><b>Lib</b> stands for <b>Libre</b> — free as in freedom, not just free as in price.</h3>

Open-source, self-hostable captcha. No tracking, no telemetry, no lock-in. Server-side proof-of-work and cryptographic verification — fully transparent. Client-side [polymorphic WASM VM](https://github.com/libcaptcha/quickjs-wasm) raises the cost of automated solving. Drop-in SDK or standard HTTP API.
